# MongoDB
1. install  
```
brew install mongodb
```
  
2. Run
```
mongod --config /usr/local/etc/mongod.conf
```
or  
```
brew services start mongodb
```
  
3. Stop
```
brew services stop mongodb
```
  
4. Restart
```
brew restart start mongodb
```
